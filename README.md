# ofAlReves
A set of OpenFrameworks scenes used for "Compagnie Al Reves", Aix en Provence.

# How to run
Scenes are managed by the great ofxAppUtils. Use function keys (F1, F2, ...) to access specific scenes.

# Addons
 - ofxGui
 - ofxOpenCv
 - ofxXmlSettings
 - ofxAppUtils
 - ofxCv
 - ofxTween
 - ofxUI
 - ofxKinect
